# CV Claude

## Démarrer le projet

```powershell
cd cv-claude
npm install
npm run dev
```

## Ajouter ta photo

- Place ton image dans `public/profile.svg` ou `public/profile.png`.
- Le code utilise `data.photo` dans `src/App.jsx`.
- Pour remplacer l'image par ta photo, remplace le fichier `public/profile.svg` par ton image et garde le même nom.

## Notes

- Le site est servi sur `http://localhost:5173/`.
- Le design utilise Tailwind, GSAP et React.
